# Brand Guidelines — Rhino Fitness

## Overview
A mobile fitness/workout tracking app centered on a rhino mascot, conveying strength, resilience, and momentum. The visual identity uses a dark theme with vivid purple accents to feel bold, energetic, and modern — suited for gym/training use in low-light or high-contrast environments.

## Logo / Mascot
- **Symbol:** Stylized rhino silhouette, side profile, charging pose.
- **Usage:** Appears centered at the bottom of key screens (e.g., Home) as a footer mark/watermark.
- **Color:** Solid brand purple on dark backgrounds.

## Color Palette

| Role | Color | Notes |
|---|---|---|
| Background (primary) | Near-black / charcoal `#1B1B1F`–`#222226` | App shell, main canvas |
| Brand Purple (primary) | Deep purple `#5B2A9E`–`#6A2FB0` | Primary buttons, key cards (Planner, Nutrition) |
| Brand Purple (light) | Lavender/orchid `#9B7FD4`–`#A88EDB` | Secondary cards (Workouts), gradients |
| Accent Gradient | Purple gradient (dark → light purple) | Progress card, Planner card backgrounds |
| Text (primary) | White `#FFFFFF` | Headlines, nav labels |
| Text (secondary/muted) | Light gray `#D0D0D5` | Day selector inactive state |
| Surface / Chip | Light gray `#E4E4E8` | Day-of-week pills (inactive) |
| Surface (active state) | Brand purple | Selected day pill (e.g., "Wed 24") |

## Typography
- **Headline / Display:** Bold, rounded sans-serif, italic-leaning weight for warmth (e.g., "Hi, Ark", "Home", "Planner", "Nutrition") — friendly but strong.
- **Body / Labels:** Same sans-serif family, regular-to-semibold weight for dates, stats, and nav items.
- **Numerals:** Bold, high-contrast — used prominently for stats (weight, calories, time, dates).

## UI Components & Patterns
- **Rounded corners:** Large border-radius (~20–24px) on cards, buttons, and pills — soft, approachable feel despite bold colors.
- **Card-based layout:** Dashboard built from modular rounded cards (Progress, Planner, Workouts, Nutrition) of varying sizes to create visual hierarchy.
- **Top nav bar:** Dark pill-shaped bar with profile icon (left), screen title (center), hamburger menu (right).
- **Day selector:** Horizontal row of rounded pills (Sun–Sat), active day highlighted in brand purple.
- **Data visualization:** Simple bar charts (Progress) and calendar grids (Planner) rendered in white/light purple on dark or purple backgrounds.
- **Icons:** Simple, solid white line/glyph icons (profile, dumbbell, flame/streak, calendar).

## Tone & Voice
- Friendly, motivational, personal — greets user by name ("Hi, Ark").
- Direct and functional copy for data-heavy screens (stats, targets, streaks).
- Encouraging without being overly casual; short, confident labels ("Progress," "Streak," "Target").

## Key Screens Referenced
1. **Sign In** — minimal, username/password, dark background.
2. **Home Dashboard** — greeting, weekly day selector, progress snapshot with photo, quick-access cards (Planner, Workouts, Nutrition).
3. **Profile** — user stats (age, start weight, goal weight, daily calories), activity history, achievements.
4. **Planner** — calendar view with category tags (Strength, Aerobic, Flexibility, Balance).
5. **Workouts** — categorized workout types (Strength, Aerobic, Flexibility, Balance, Custom).
6. **Nutrition** — daily calorie target vs. consumed, meal breakdown (Breakfast/Lunch/Dinner), hydration tracker.
7. **Progress** — streak tracker across the week, time-based metrics.

## Design Principles
- **Dark-first UI** for reduced eye strain and a premium, gym-ready aesthetic.
- **Purple as the emotional core** — used consistently across CTAs, highlights, and the mascot to unify the experience.
- **Big, legible numbers** for at-a-glance tracking of weight, calories, and time.
- **Consistent rounding and spacing** to keep a dense dashboard feeling calm and organized.
